# Productos Innovadores


## Descripción

Sitio web desarrollado con HTML5 y CSS3 aplicando buenas prácticas de SEO técnico.


## SEO Implementado


- Archivo robots.txt configurado.
- Sitemap.xml creado.
- Etiquetas canonical agregadas.
- Redirecciones 301 configuradas.
- Imágenes optimizadas en formato WebP.
- Textos alternativos ALT personalizados.
- Optimización de accesibilidad.


## Tecnologías utilizadas


- HTML5
- CSS3
- SEO Técnico
